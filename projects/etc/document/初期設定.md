## 初期設定

### 物件をクローンする
#### 初回
1. 任意の場所にproductionフォルダを作成する
2. コマンドプロンプトを起動しproductionフォルダにcdする
3. 以下コマンドを実行する
``` cmd
git clone https://github.com/ryota-tojo/home_server_operation.git
```
#### 2回目以降(masterブランチをチェックアウト＋更新)
``` cmd
git fetch origin
git checkout master
git pull origin master
```
### 必要なライブラリをインストールする
`projects/etc/init/pip_tools.bat` を実行する
